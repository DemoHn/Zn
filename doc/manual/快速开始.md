## 变量的声明

在计算机编程领域，变量用于标识某个存储的值或者内容；这里的「变量」类似于数学函数里面的「变量」，亦即「可变的量」。变量的值可以随时读取，亦可以通用「赋值」操作修改其对应的值。

在 Zn 中，所有变量必须 **先声明后使用**：

```zn
令生产总值 = 120

生产总值 = 200
```

如上面的代码所示，第1行的 `令生产总值 = 120` 即是声明变量： 其中，`令` 是 Zn 中的一个关键字，用于声明其后的变量名；在变量名后面有个 `=`，用于给定此变量的初值为 `120`。

第3行中的 `生产总值 = 200` 即是将 `生产总值` 这个变量的值从原来的 `120` 改为现在的 `200`。

> 注意：Zn 使用中文编程，关键字及变量名称都是中文，且关键字和变量名之间无需空格；所以在变量命名时，需要适当避开 `令`，`如果`，`每当` 等关键字；如果实在需要命名诸如 `将军令` 的变量，需要在变量名前后各加一个 `·`，变成 `·将军令·` 

## 变量的基本类型

Zn 的变量支持以下几种基本类型：

```zn
令数值* = 123.45
令文本* =「千里之行，始于足下。」 注：「也可以使用此引号 “ ” 来包围文本」
令布尔值* = 真                 注：「布尔类型有两个值，一为 「真」，一为「假」」

令数组* =【1，3，5，7，9】
令列表* =【
    名称 ==「清华大学」         注：中间的符号是两个「等于」号
    分数线 == 762
    录取要求 ==【「文科」，「理科」】
】

令空值* = 空
```

上面的代码分别列举出了 Zn 所支持的几种基本类型：`数值`、`文本`、`布尔`、`数组`、`列表` 以及特殊的 `空` 类型. 每种类型都有其特有的处理方法，具体的用法将会在后文说明。

> 注：`+`、`-`、`*`、`/` 可以参与到变量名的命名当中，如 `数值*` 是一个符合要求的变量名.

## 数值的四则运算

和大多数编程语言不同，`+`、`-`、`*`、`/` 并不是关键字，亦不参与到四则运算当中。数据的四则运算采用「声明」的形式，如下所示：

```zn
令加法结果 =【120，80】之和         注：「120 + 80 = 200」
令减法结果 =【15，8，6】之差        注：「15 - 8 - 6 = 1」
令乘法结果 =【27.47，2.09】之积     注：「27.47 * 2.09 = 57.4123」
令除法结果 =【48，3】之商           注：「48 / 3 = 16」
```

其中，用`【 】` 括住的是一个数组，里面的每一个元素都是数值。数组需要至少一个元素，其中第一个元素为被加数/被减数，后面的元素依次对前面的结果进行相加/相减/相乘/相除。`之` 是一个关键字（可以用 `的` 替代），后面的 `和/差/积/商` 分别代表着这个数组的属性。显然， `xx 之和` 得到的结果即是数组内各个元素之和，对于其他属性以此类推。

同时对于数值，也可以用属性及方法对于数值 `+1` 或 `-1`：

```zn
令数值 = 8.5
令X1 = 数值之+1         注：X1 = 8.5 + 1 = 9.5，等同于 令X1 =【数值，1】之和
令X2 = 数值之-1         注：X2 = 8.5 - 1 = 7.5，等同于 令X1 =【数值，1】之差

令Y = 数值之（+1）       注：数值的值从 8.5 加1到 9.5；并将值传递给 Y
令Z = Y之（-1）         注：数值的值从 9.5 减1到 8.5；并将值传递给 Z
```

> 注：数值的属性仅限于 `+1`, `-1`；并没有 `+2`、`-2`、`*2` 这样的属性！

## 流程控制

Zn 支持 `如果`、`每当` 两种流程控制语句。

- `如果` 语句用于控制具体执行哪一段代码，如果设定条件的值为 `真`，那么执行 `如果` 语句下的语句块；否则不执行。具体逻辑如下所示：

```zn
如果【15，6】之和 > 20：
    令文本 = 「15 + 6 > 20」
    （显示：文本）
否则：
    （显示：「15 + 6 <= 20」）
```

在上面的程序中，第1行指定了一个 `【15，6】之和 > 20` 的表达式；这个表达式的结果只会有「真」和「假」两种可能。如果表达式的结果为「真」，那么即执行「如果」语句下，第2行和第3行的程序；如果是「假」，那么执行「否则」语句下，第5行的程序。

显然，15 + 6 = 21 > 20，因此程序最终会在终端显示「15 + 6 > 20」。

> 注1：Zn 以缩进区分语句块。一个缩进单元为 4个空格 或者 1个Tab。同一个文件里的所有缩进类型要么都是空格，要么都是Tab，不能空格和Tab混用。后面的所有语句亦然。
> 注2：「否则」语句可以省略，但是「如果」语句下必须要另起一行，并添加一个缩进单元编写里面的逻辑。
> 注3：（显示：xx） 是 Zn 里面的一个内置方法，调用此方法可以在终端里面打印出变量的值. 

- `每当` 语句可以重复执行对应的语句块，直到指定条件不再满足为止，如下所示：

```zn
令倒计时 = 10
每当倒计时 > 0：
    （显示：「倒计时：」、倒计时）
    倒计时之（-1）
（显示：「倒计时结束」）
```

在上面的程序中，只要第2行中的 `倒计时 > 0` 条件成立，「每当」语句后面的语句块（第3行，第4行）都会从头到尾执行一遍；并且每执行一次，变量 `倒计时` 的值都会减1；当执行第10遍结束时，`倒计时`的值为0，因此不符合条件，跳出循环，执行第5行。

- 类似功能的还有 `遍历` 语句：此语句将指定数组或者列表从头到尾遍历一遍其中的每一个元素，并执行对应的语句块，如下所示：

```zn
令数列 =【4，8，16，32，64，128，256】
令结果 = 0
以元素遍历数列：
    结果为【元素，结果】之和

（显示：结果）
```

在上面的程序中，`遍历` 语句声明了一个 `元素` 的变量，此变量的值随着每一次循环依次为 4，8，16，... 直到 256 为止。在每次循环时，`结果` 的值都会和当前元素的值进行累加，在遍历结束后跳出循环，显示最终结果。实际上上面的程序即是对于数列的元素求和。
