## 变量的声明

在计算机编程领域，变量用于标识某个存储的值或者内容；这里的「变量」类似于数学函数里面的「变量」，亦即「可变的量」。变量的值可以随时读取，亦可以通用「赋值」操作修改其对应的值。

在 Zn 中，所有变量必须 **先声明后使用**：

```zn
令生产总值 = 120

生产总值 = 200
```

如上面的代码所示，第1行的 `令生产总值 = 120` 即是声明变量： 其中，`令` 是 Zn 中的一个关键字，用于声明其后的变量名；在变量名后面有个 `=`，用于给定此变量的初值为 `120`。

第3行中的 `生产总值 = 200` 即是将 `生产总值` 这个变量的值从原来的 `120` 改为现在的 `200`。

> 注意：Zn 使用中文编程，关键字及变量名称都是中文，且关键字和变量名之间无需空格；所以在变量命名时，需要适当避开 `令`，`如果`，`每当` 等关键字；如果实在需要命名诸如 `将军令` 的变量，需要在变量名前后各加一个 `·`，变成 `·将军令·` 

## 变量的基本类型

Zn 的变量支持以下几种基本类型：

```zn
令数值* = 123.45
令文本* =「千里之行，始于足下。」 注：「也可以使用此引号 “ ” 来包围文本」
令布尔值* = 真                 注：「布尔类型有两个值，一为 「真」，一为「假」」

令数组* =【1，3，5，7，9】
令列表* =【
    名称 ==「清华大学」         注：中间的符号是两个「等于」号
    分数线 == 762
    录取要求 ==【「文科」，「理科」】
】

令空值* = 空
```

上面的代码分别列举出了 Zn 所支持的几种基本类型：`数值`、`文本`、`布尔`、`数组`、`列表` 以及特殊的 `空` 类型. 每种类型都有其特有的处理方法，具体的用法将会在后文说明.
