## 关于本手册

本手册包含了 Zn 语言的语法结构、类型定义、执行流程及内置标准库等内容，希望能够向使用 Zn 语言的用户深入了解其功能及使用方式以供参考及后续开发。

本手册并不是严格的语言标准 (specification)，更像是介于入门教程和语法标准的参考文档。由于 Zn 语言正处于早期开发阶段，关于此语言的特性变更将会十分频繁；故而当发现参考手册与具体实现有冲突的地方时，请暂且以具体实现为准。
